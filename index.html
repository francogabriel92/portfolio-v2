GymApp

One page web-app dedicated to the management of gym clients through a CRUD system. The user interface was made with ReactJs and React Router.

The backend is designed with NodeJs and Express, has a login system with Json Web Token technology, tests performed with Jest and the database is based on MongoDB.

Techs and frameworks used: ReactJs, React Router, React Bootstrap, Axios, Mongoose, Formik, Yup, MongoDb, JWT, Jest, Bcrypt.
